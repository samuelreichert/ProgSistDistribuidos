## Para rodar os códigos fontes

### Java
* Prerequisito: ter o java JDK instalado
* Abra o terminal e rode o comando:
* `java 'nome_do_arquivo'`

### Ruby
* Prerequisito: Ter o ruby instalado
* Abra o terminal e rode o comando:
* `ruby 'nome_do_arquivo.rb'`
